How to connect on Multiplayer

Windows

This tutorial you'll need to use CMD or PowerShell!

Specify the fnaf doom directory

cd C:\path\to\path

Example:

cd C:\Downloads\fnaf-doom\

Connecting on the server:

.\gzdoom.exe +map e1m2 -join Set-Ip-Address -file FiveNightsAtFreddys1 -file FNAF.pk3 -file brightmaps.pk3 -file FiveNightsAtFreddys1_3.wad

Note: You can create a script.bat to run the game!

Linux

Terminal

You can open the terminal on fnaf directory without specify the directory

Specify the fnaf doom directory if you want:

cd ~/path/to/path

Example:

cd ~/Downloads/fnaf-doom/

Creating a script to connect on multilplayer:

touch connect.sh

Adding the command into the connect.sh

echo "gzdoom +map e1m2 -join Set-Ip-Address -file FiveNightsAtFreddys1 -file FNAF.pk3 -file brightmaps.pk3 -file FiveNightsAtFreddys1_3.wad" > connect.sh

Connecting on the server:

./connect.sh
